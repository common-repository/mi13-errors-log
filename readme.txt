Plugin for monitoring errors on your website through the admin panel.

== Description ==

This plugin displays the last 10 errors of your site in the wp admin panel.

== Installation ==

1. Copy the plugin folder <strong>mi13-errors-log</strong> to <strong>/wp-content/plugins/</strong>. 
2. Activate the plugin via the <strong> menuPlugins</strong>.
3. That's it.
